Font Sizing Examples

* D1. Default (root) font size (16px)

This is text with the default font size for this webpage, which is set to 1em (or, 16px)
Note: set in the body (root element) rule on the master css and inherited rule from the p rule on the reset css

* D2. This section's default font size (24px)

This is the font size of this section [in light green], which is set to 1.5em
(or, 1.5 * 16px (closest ancestor element, body) = 24px)
Note: set in the section element (id, font-example-section) rule on the units css

* 1. px (1/96 dots per inch)

This is text with a font size set as 8px

This is text with a font size set as 16px

This is text with a font size set as 32px

* 2. em (inherits the font size set from its closest ancestor element)

This is text with a font size set as .5em
(.5 x 24px caculates to 12px)

This is text with a font size set as 1em
(1 x 24px caculates to 24px)

This is text with a font size set as 2em
(2 of 24px caculates to 48px)

* 3. rem (inherits font size from root element)

This is text with a font size set as .5rem
(.5 x 16px caculates to 8px)

This is text with a font size set as 1rem
(1 x 16px caculates to 16px)

This is text with a font size set as 2rem
(2 x 16px caculates to 32px)

This is text with a font size set as 1vw

This is text with a font size set as 2vw

This is text with a font size set as 3vw

This is text with a font size set as 4vw

Width Sizing Examples

The box with black border is the outer parent container
The box with the red border is the child container

* 1. px

1p. parent is set to 600px

1a. child is set to 200px

1b. child is set to 400px

1c. child is set to 600px

1d. child is set to 700px

* 2. percent % (percentage of its containing (parent) element)

2p. parent is set to 50% of its parent element

2a. child is set to 25% of its parent element

2b. child is set to 75% of its parent element

2c. child is set to 100% of its parent element

2d. child is set to 120% of its parent element

* 3. vw (viewport width - relative to 1% of viewport width)

3p. parent is set to 50vw (50% the width of the viewport)

3a. child is set to 25vw (25% of the width of the viewport

3b. child is set to 75vw (75% of the width of the viewport

3c. child is set to 100vw (100% of the width of the viewport

3d. child is set to 120vw (120% of the width of the viewport
